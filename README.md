# SRI RAM MOTORS - Authorised Service Station Website

A modern, responsive website for SRI RAM MOTORS automobile service station.

## Setup Instructions

1. **Add the Logo Image:**
   - Place the Maruti Suzuki logo image in the project folder
   - Name it `maruti-suzuki-logo.png`
   - The logo should be placed in the same directory as `index.html`

2. **Open the Website:**
   - Simply open `index.html` in your web browser
   - Or use a local server for better performance
